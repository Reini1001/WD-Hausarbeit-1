Introduction

What is SVG?

Scalable Vector Graphics (SVG) is a tool used to realise two-dimensional images. There are two types of SVG graphics, .SVG files and usages inside HTML, this site will focus on the latter. SVG Graphics are used because of multiple factors.

  • Space - SVG files are resolution independent, because SVG only describes how the picture is made.
  • Speed - In both Processing and Transmission speed SVG is superior to other file formats.
  • Scalability - SVG files can be rendered in any resolution wanted. SVG is a lossless format.
