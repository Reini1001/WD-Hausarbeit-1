Styling

SVG's consist of content-elements (e.g., text, lines, and shapes). How this content is rendered is mostly defined through styling properties. SVG is based on XML and shares a lot of properties with CSS2/CSS3.
This side shows some examples of styling-properties in HTML and CSS3.

Examples

Font-size

The font-size property determines the size of the font.

The value can be an absolute-size, relative-size or length-percentage. The default value is "medium".

The font-size property is animatable.

You can use this property in the elements:
<altGlyph>,
<text>,
<textPath>,
<tref>
and <tspan>.

HTML:
<svg>
<textid="font-size-demo"x="50"y="80"font-size="0.5rem">test text</text>
</svg>

CSS:
#font-size-demo{
font-size:0.5rem;
}

Direction

The direction property specifies the direction in witch the Text propagates.

The value can be ltr (left to right) or rtl (right to left). The default value is ltr.

The direction property is animatable.

You can use this property in the elements:
<altGlyph>,
<text>,
<textPath>,
<tref>
and <tspan>.

HTML:
<svg>
<textid="direction-demo"x="50%"y="75%"direction="rtl">test text</text>
</svg>

CSS:
#direction-demo{
direction:ltr;
}

Fill

The fill property has two different use-cases.
1. Color
In this use-case the fill property specifies the color of Text and shapes.

It's values can be a color. The default value is black.
And it is animatable.

In this use-case the property can be used in thees elements:
<altGlyph>,
<circle>,
<ellipse>,
<path>,
<polygone>,
<ployline>,
<rect>,
<text>,
<textPath>,
<tref>
and <tspan>.

2. Animation
In this use-case the fill property defines the state of the animation

It's values can be freeze or remove. The default value is remove.
And it is NOT animatable.

In this use-case the property can be used in thees elements:
<animate>,
<animateColor>,
<animateMotion>,
<animateTransform>,
and <set>

The visual and code examples are for the first use-case.

HTML:
<svg>
<rectid="fill-demo"x="25%"y="25%"width="50%"height="50%"fill="black"/>
</svg>

CSS:
#fill-demo{
fill:black;
}

Fill-opacity

The fill-opacity property defines the opacity of colors.

The value can be numbers from 0 to 1 or percentage numbers. The default value is 1 (100%).

The fill-opacity property is animatable.

You can use this property in the elements:
<altGlyph>,
<circle>,
<ellipse>,
<path>,
<polygone>,
<ployline>,
<rect>,
<text>,
<textPath>,
<tref>
and <tspan>.

HTML:
<svg>
<rectid="fill-opacity-demo"x="25%"y="25%"width="50%"height="50%"fill="black"fill-opacity="50%"/>
</svg>

CSS:
#fill-opacity-demo{
fill-opacity:50%;
}

Color

The color property provides a color for other properties like fill, stroke and stop-color.

The value can be a color or inherit. The default value is user agent dependent.

The color property is animatable.

You can use this property in any element

HTML:
<svg>
<gid="color-demo"color="red">
<rectx="10%"y="25%"width="50px"height="50px"fill="currentcolor"/>
</g>
</svg>

CSS:
#color-demo{
color:red;
}

Writing-mode

The writing-mode property specifies the inline-progression-direction.

The value can be horizontal-tb | vertical-rl | vertical-lr. The default value is horizontal-tb.

The writing-mode property is animatable.

You can use this property in the elements:
<altGlyph>,
<text>,
<textPath>,
<tref>
and <tspan>.

HTML:
<svg>
<textid="writing-mode-demo"x="10%"y="10%"writing-mode="tb">test text</text>
</svg>

CSS:
#writing-mode-demo{
writing-mode:tb;
}
