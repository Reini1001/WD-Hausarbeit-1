Painting

Painting operations allow for adjusting visuals of svg-elements like basic shapes, paths and texts. They are specified as properties of the element within inline-svg. This Page will outline the basic painting operations.

Properties

fill:
stroke:

Code

<svg viewBox="0 0 120 120">
 <circle cx="60" cy="60" r="50" fill= "white" "black" "#289819" "#3faef8" stroke= "white" "black" "#289819" "#3faef8" stroke-width="10"/>
 </svg>

Description

The 'fill' property determines the inner color of the element it is applied on.
You can use a predefined color like "white" and "black", or give any color in hexadecimal rgb format (eg. "#ee23f0")

The 'stroke' property determines the border or line color of the element it is applied on.
You can use a predefined color like "white" and "black", or give any color in hexadecimal rgb format (eg. "#ee23f0")
