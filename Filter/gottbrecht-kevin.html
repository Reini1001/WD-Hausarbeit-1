SVG Filters

SVG Filters are a simple way to style elements in more ways than possible with CSS. This small tutorial will show usecases in both SVG and uses in HTML. SVG filters mostly find their use in Text and Background styling.

  1. Usage of filters in Graphics
  2. Usages in Text
A base structure is needed to use SVG filters, one example is the following: 
<svg>
<filter id="filterName">
    <filterType filterArgs=""></filterType>
</filter>
</svg>
A list full of Filter types and arguments can be found here! Those filters can be used in CSS as follows: 
 #HTMLfilter
  {
filter: url(#filterName)
 }

To apply filters to the examples shown, simply hover over them!

Use in Graphics

This example will show how SVG filters can also be applied to SVG graphics. This (very simple) tree changes color on hover. The color change is based on the feColorMatrix filter. This filter appies changes based on a matrix containing the R, G, B and Alpha channel.

Filters can also be applied to img tag integrated pictures. The blur filter could be used to hide content. In this case hovering over the picture removes the filter. This feGaussianBlur filter allows modification of the distortion by changing the filters arguments
