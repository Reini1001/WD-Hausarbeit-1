SVG Filters

SVG filters can modify both SVG and HTML parts of a Website. The following tutorial will show two showcases.

  1. Usage of filters in SVG elements
  2. Usage of filters in HTML shown by <img> tag pictures
A base structure is needed to use SVG filters, one example is the following: 
           <svg>
    <filter id="filterName">
        <filterType filterArgs=""></filterType>
    </filter>
            </svg>
A list full of Filter types and arguments can be found here! Those filters can be used in CSS as follows: 
            #HTMLfilter:hover
            {
    filter: url(#filterName)
            }
